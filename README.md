### 第2章 算法基础
#### 2.1 插入排序
**插入排序：**对于少量的元素的排序，是一个比较有效的算法。插入排序排序有点想对扑克牌进行有顺序的排序。<br />
```
void insertionSort (int a[], int len)
{
    int i, j, data;
    for ( i = 1; i < len; i++)
    {
        data = a[i];
        j = i - 1;
        while ( j >= 0 && a[j] > data)
        {
            a[j + 1] = a[j];
            j--;
        }
        a[j + 1] = data;
    }
}
```
**循环不变式与插入排序的正确性** <br />
